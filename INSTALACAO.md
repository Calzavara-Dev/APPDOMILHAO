# 🚀 Instalação Rápida - Análise de Pares

## Passos para executar o app:

### 1. Instalar dependências
```bash
npm install
```

### 2. Executar em desenvolvimento
```bash
npm run dev
```

### 3. Acessar no navegador
```
http://localhost:3000
```

### 4. Para smartphone:
- Acesse o mesmo endereço no seu celular
- Ou instale como PWA (app nativo)
